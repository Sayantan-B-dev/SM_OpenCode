# Course Structure

This guide is organized into 18 sections that take you from complete beginner to confident OpenCode user. Each section corresponds to a lesson in the video course.

## What You Will Learn

Here is everything this course covers:

| # | Section | What You Will Learn |
|---|---------|---------------------|
| 1 | Welcome | Why OpenCode matters and how to use this guide |
| 2 | Course Structure | Overview of all topics (this page) |
| 3 | About OpenCode | What OpenCode is and why it is better |
| 4 | Prerequisites | What you need to know before starting |
| 5 | Setting Up OpenCode | Install on any platform step by step |
| 6 | Core Commands | Every essential command explained |
| 7 | TUI and Interfaces | Terminal UI, Desktop App, Web UI, IDE |
| 8 | Free Setup (Go & Zen) | Configure free providers |
| 9 | Models | Understanding and switching between models |
| 10 | Agents and Subagents | Create and customize AI agents |
| 11 | Skills & Automations | Build reusable workflow automations |
| 12 | Building an Application | Complete app from scratch demo |
| 13 | Comparing Models | Test multiple models side by side |
| 14 | MCP Server Setup | Connect external tools via MCP |
| 15 | Local Ollama & OpenRouter | Run models locally and via OpenRouter |
| 16 | Context & Cost Optimization | Manage context windows and save money |
| 17 | GitHub, LSP & ACP | Integrations with your dev tools |
| 18 | Conclusion | Summary and next steps |

## Prerequisites Flow

Before diving in, make sure you have covered section 4 (Prerequisites). You should be comfortable with:

- Basic JSON structure
- JavaScript / TypeScript fundamentals
- Node.js and npm
- Using a terminal
- Git basics

## Time Estimate

Each section takes approximately 5-15 minutes to read and follow along. The total course takes about 2-3 hours to complete including hands-on practice.
