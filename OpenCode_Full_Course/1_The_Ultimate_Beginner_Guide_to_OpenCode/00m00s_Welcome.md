# Welcome

Welcome to **The Ultimate Beginner Guide to OpenCode** — your complete starting point for learning and mastering OpenCode, the open-source alternative to GitHub Copilot Codex and Anthropic Claude Code.

## Why Use OpenCode?

OpenCode is not just another AI coding tool. It is a **provider-agnostic, agent-based, fully open-source** coding assistant that puts you in control. Here is why developers are switching to OpenCode:

| Reason | Explanation |
|--------|-------------|
| **Open Source** | 100% Apache 2.0 licensed. Inspect, modify, and distribute the code freely. |
| **Provider Agnostic** | Use any LLM provider — OpenAI, Anthropic, Google, Mistral, Ollama, Zen, or your own custom endpoint. |
| **Agent-Based** | Specialized agents (General, Explore, Scout) handle different types of tasks. |
| **Multi-Interface** | Use CLI, TUI, Web UI, or API — the choice is yours. |
| **Full Undo/Redo** | Every action is reversible. Experiment without fear. |
| **Cost Control** | Switch between cheap and expensive models per task to optimize spending. |
| **MCP Support** | Connect to databases, APIs, and services via the Model Context Protocol. |
| **Local First** | Works fully offline with local models via Ollama. |

## What You Will Gain

By the end of this guide, you will be able to:

- Install and configure OpenCode on any platform.
- Navigate the TUI and use all core commands.
- Set up and switch between multiple AI models.
- Create custom agents and subagents.
- Build skills and workflow automations.
- Connect MCP servers for external tools.
- Optimize context windows and control costs.
- Integrate OpenCode with GitHub and your IDE.
