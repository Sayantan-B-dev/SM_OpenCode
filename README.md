# OpenCode Learning Repository

A comprehensive OpenCode learning repository with notes, tutorials, setup guides, workflows, MCP integrations, skills, agents, sub-agents, model comparisons, context management, cost optimization, Ollama/OpenRouter setups, and practical examples from beginner to advanced real-world use cases.

## Contents

- OpenCode setup and configuration
- Agents and sub-agents
- Skills and workflow automation
- MCP server integration
- Model comparisons and provider setup
- Ollama and OpenRouter guides
- Context management and cost optimization
- Real-world examples and project walkthroughs

## Purpose

This repository serves as a structured knowledge base for learning and mastering OpenCode through curated notes and hands-on examples.
